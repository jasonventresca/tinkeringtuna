# Hello World - NuxtJS
An example of how to deploy a **NuxtJS, thanks to Kinsta.

## Local development
```bash
npm run dev
```

## Production
Push to the `main` branch. The CD setup in GitHub Actions should rebuild and publish the result in ~ under 10 seconds.
